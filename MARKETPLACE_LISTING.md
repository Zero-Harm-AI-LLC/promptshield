# PromptShield — GitHub Marketplace Listing

## Short description

Detect prompt injection, PII leaks, and unsafe LLM usage in pull requests.

## Long description

PromptShield is a free AI security reviewer for pull requests.

It helps developers catch AI-specific security risks before code is merged, including:

- prompt injection risks
- PII sent to LLMs
- prompt logging
- secrets exposure
- unsafe tool usage
- data sent from databases or documents into LLM context

Unlike traditional AppSec tools, PromptShield focuses specifically on **LLM security patterns**.

## Best for

- AI startups
- teams building LLM applications
- developers integrating OpenAI, Anthropic, Gemini, or similar APIs
- security teams reviewing AI features
